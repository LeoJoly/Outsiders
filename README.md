# Outsiders

Outsiders is a community plateform that helps outdoor sports passionates to be connected to share trips and sport sessions.

## General informations

### Back technologies

- Node.js
- Express
- PosgreSQL
- Sqitch
- AWS

### Front technologies

- React
- Redux
- React-router
- Scss
- Axios
- gsap

## Start

In the terminal :
```bash
cd outsiders-front
```

then run :
```bash
yarn
```

then run :
```bash
yarn start
```
